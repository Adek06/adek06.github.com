Trigonometry Function¶

Before comprehend Fourier Series,we have to kown what Trigonometry Function is first.

Like $1,cosx,cos2x,con3x...sinx,sin2x,sin3x...$,thoes are called trigonometry function.

In [1]:
import matplotlib.pyplot as plt
import numpy as ny
plt.style.use('seaborn-whitegrid')
x = ny.linspace(-4*ny.pi,4*ny.pi,1000)
plt.plot(x,ny.cos(x))
plt.plot(x,ny.sin(x))
plt.show()

we can get two properties:

1.They are always $2\pi$ cycle one time.So,integral $[-\pi,\pi]$ always equal 0.

$$\int_{-\pi}^{\pi}cosnx dx =\int_{-\pi}^{\pi}sinnx dx = 0$$ $$\int_{-\pi}^{\pi}cosnxcosmx dx =\int_{-\pi}^{\pi}sinnxsinmx dx = 0$$ $$\int_{-\pi}^{\pi}cosnxsinmdx =0$$

2.But integral of the square of trigonometry functions isn't equal 0.

$$\int_{-\pi}^{\pi}cos^2nx dx =\int_{-\pi}^{\pi}sin^2nx dx = \pi$$

Fourier Series¶

Now we assume a function like this.

$$f(x) = \frac{a_0}{2}+\sum_{k=1}^{\infty}(a_kcoskx dx+b_ksinkx dx)$$

We don't kown what $a_0,a_k,b_k$ is. So let's get them.

get $a_0$¶

$$\int_{-\pi}^{\pi}f(x)dx = \frac{a_0}{2}\int_{-\pi}^{\pi}dx+\sum_{k=1}^{\infty}(a_k\int_{-\pi}^{\pi}coskx dx+b_k\int_{-\pi}^{\pi}sinkx dx)$$

Because of properties one.

$$\dfrac{1}{\pi}\int_{-\pi}^{\pi}f(x)dx = a_0$$

get $a_n$¶

$$\int_{-\pi}^{\pi}f(x)cosnx dx = \frac{a_0}{2}\int_{-\pi}^{\pi}cosnxdx+\sum_{k=1}^{\infty}(a_k\int_{-\pi}^{\pi}coskxcosnx dx+b_k\int_{-\pi}^{\pi}sinkxcosnx dx)$$

Because of properties one and two.

$$\dfrac{1}{\pi}\int_{-\pi}^{\pi}f(x)cosnxdx = a_n$$

get $b_n$¶

$$\int_{-\pi}^{\pi}f(x)sinnx dx = \frac{a_0}{2}\int_{-\pi}^{\pi}sinnx dx+\sum_{k=1}^{\infty}(a_k\int_{-\pi}^{\pi}coskxsinnx dx+b_k\int_{-\pi}^{\pi}sinkxsinnx dx)$$

Because of properties one and two.

$$\dfrac{1}{\pi}\int_{-\pi}^{\pi}f(x)sinnxdx = b_n$$
